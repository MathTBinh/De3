Chủ Đề 1: Phương Trình và Bất Phương Trình

Tác giả: Thạc sĩ Nguyễn Thanh Bình

A. Kiến Thức Cần Nhớ

I. Phương Trình Lượng Giác

1. Phương trình lượng giác cơ bản

a) Phương trình \(\sin x = m\):

  • Nếu \(|m| > 1\), phương trình vô nghiệm.
  • Nếu \(|m| \leq 1\), gọi \(\alpha \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\) sao cho \(\sin \alpha = m\). Khi đó: \[ \sin x = m \Leftrightarrow \sin x = \sin \alpha \Leftrightarrow \begin{cases} x = \alpha + k2\pi \\ x = \pi - \alpha + k2\pi \end{cases} (k \in \mathbb{Z}). \]
  • Trường hợp đặc biệt:
    • \(\sin x = 1 \Leftrightarrow x = \frac{\pi}{2} + k2\pi\).
    • \(\sin x = -1 \Leftrightarrow x = -\frac{\pi}{2} + k2\pi\).
    • \(\sin x = 0 \Leftrightarrow x = k\pi\).
  • Nếu \(x\) đo bằng độ: \[ \sin x = \sin a^\circ \Leftrightarrow \begin{cases} x = a^\circ + k360^\circ \\ x = 180^\circ - a^\circ + k360^\circ \end{cases} (k \in \mathbb{Z}). \]

b) Phương trình \(\cos x = m\):

  • Nếu \(|m| > 1\), phương trình vô nghiệm.
  • Nếu \(|m| \leq 1\), gọi \(\alpha \in [0, \pi]\) sao cho \(\cos \alpha = m\). Khi đó: \[ \cos x = m \Leftrightarrow \cos x = \cos \alpha \Leftrightarrow \begin{cases} x = \alpha + k2\pi \\ x = -\alpha + k2\pi \end{cases} (k \in \mathbb{Z}). \]
  • Trường hợp đặc biệt:
    • \(\cos x = 1 \Leftrightarrow x = k2\pi\).
    • \(\cos x = -1 \Leftrightarrow x = \pi + k2\pi\).
    • \(\cos x = 0 \Leftrightarrow x = \frac{\pi}{2} + k\pi\).
  • Nếu \(x\) đo bằng độ: \[ \cos x = \cos a^\circ \Leftrightarrow \begin{cases} x = a^\circ + k360^\circ \\ x = -a^\circ + k360^\circ \end{cases} (k \in \mathbb{Z}). \]

c) Phương trình \(\tan x = m\):

  • Gọi \(\alpha \in \left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\) sao cho \(\tan \alpha = m\). Khi đó: \[ \tan x = m \Leftrightarrow \tan x = \tan \alpha \Leftrightarrow x = \alpha + k\pi (k \in \mathbb{Z}). \]
  • Nếu \(x\) đo bằng độ: \[ \tan x = \tan a^\circ \Leftrightarrow x = a^\circ + k180^\circ (k \in \mathbb{Z}). \]

d) Phương trình \(\cot x = m\):

  • Gọi \(\alpha \in (0, \pi)\) sao cho \(\cot \alpha = m\). Khi đó: \[ \cot x = m \Leftrightarrow \cot x = \cot \alpha \Leftrightarrow x = \alpha + k\pi (k \in \mathbb{Z}). \]
  • Nếu \(x\) đo bằng độ: \[ \cot x = \cot a^\circ \Leftrightarrow x = a^\circ + k180^\circ (k \in \mathbb{Z}). \]

2. Phương trình lượng giác đưa về dạng cơ bản

  • \(\sin f(x) = \sin g(x) \Leftrightarrow \begin{cases} f(x) = g(x) + k2\pi \\ f(x) = \pi - g(x) + k2\pi \end{cases} (k \in \mathbb{Z})\).
  • \(\cos f(x) = \cos g(x) \Leftrightarrow \begin{cases} f(x) = g(x) + k2\pi \\ f(x) = -g(x) + k2\pi \end{cases} (k \in \mathbb{Z})\).
  • Đối với các phương trình dạng \(\sin^2 u(x) = \sin^2 v(x)\), \(\cos^2 u(x) = \cos^2 v(x)\), \(\sin^2 u(x) = \cos^2 v(x)\), dùng công thức hạ bậc để đưa về dạng \(\cos f(x) = \cos g(x)\).
  • Một số phương trình lượng giác có thể dùng công thức lượng giác và biến đổi để đưa về dạng tích \(A(x) \cdot B(x) = 0\).

II. Phương Trình, Bất Phương Trình Mũ và Lôgarit

1. Phương trình mũ

  • Với \(a > 0\), \(a \neq 1\):
    • \(a^{f(x)} = b \Leftrightarrow f(x) = \log_a b\) (với \(b > 0\)).
    • \(a^{f(x)} = a^{g(x)} \Leftrightarrow f(x) = g(x)\).

2. Phương trình lôgarit

  • Với \(a > 0\), \(a \neq 1\):
    • \(\log_a f(x) = b \Leftrightarrow f(x) = a^b\).
    • \(\log_a f(x) = \log_a g(x) \Leftrightarrow \begin{cases} f(x) = g(x) \\ f(x) > 0 \text{ hoặc } g(x) > 0 \end{cases}\).

3. Bất phương trình mũ

  • Với \(a > 0\), \(a \neq 1\), xét bất phương trình \(a^{f(x)} > b\):
    • Nếu \(b \leq 0\), tập nghiệm là tập xác định của \(f(x)\).
    • Nếu \(b > 0\), \(a > 1\), thì bất phương trình tương đương \(f(x) > \log_a b\).
    • Nếu \(b > 0\), \(0 < a < 1\), thì bất phương trình tương đương \(f(x) < \log_a b\).
  • Xét bất phương trình \(a^{f(x)} > a^{g(x)}\):
    • Nếu \(a > 1\), thì bất phương trình tương đương \(f(x) > g(x)\).
    • Nếu \(0 < a < 1\), thì bất phương trình tương đương \(f(x) < g(x)\).

4. Bất phương trình lôgarit

  • Với \(a > 0\), \(a \neq 1\), xét bất phương trình \(\log_a f(x) > b\):
    • Nếu \(a > 1\), thì bất phương trình tương đương \(f(x) > a^b\).
    • Nếu \(0 < a < 1\), thì bất phương trình tương đương \(0 < f(x) < a^b\).
  • Xét bất phương trình \(\log_a f(x) > \log_a g(x)\):
    • Nếu \(a > 1\), thì bất phương trình tương đương \(f(x) > g(x) > 0\).
    • Nếu \(0 < a < 1\), thì bất phương trình tương đương \(0 < f(x) < g(x)\).

B. Một Số Ví Dụ

Dạng 1: Câu trắc nghiệm nhiều phương án lựa chọn

Ví dụ 1: Nghiệm của phương trình \(\sin \left(x + \frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2}\) là:

  • A. \(x = -\frac{2\pi}{3} + k2\pi\) và \(x = \pi + k2\pi (k \in \mathbb{Z})\).
  • B. \(x = -\frac{\pi}{3} + k2\pi\) và \(x = \frac{\pi}{3} + k2\pi (k \in \mathbb{Z})\).
  • C. \(x = k2\pi\) và \(x = \pi + k2\pi (k \in \mathbb{Z})\).
  • D. \(x = -\frac{\pi}{2} + k2\pi\) và \(x = \frac{5\pi}{3} + k2\pi (k \in \mathbb{Z})\).

Giải:

Do \(\sin \left(-\frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2}\), nên:

\[ \sin \left(x + \frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2} \Leftrightarrow \sin \left(x + \frac{\pi}{3}\right) = \sin \left(-\frac{\pi}{3}\right) \Leftrightarrow \begin{cases} x + \frac{\pi}{3} = -\frac{\pi}{3} + k2\pi \\ x + \frac{\pi}{3} = \pi - \left(-\frac{\pi}{3}\right) + k2\pi \end{cases} \Leftrightarrow \begin{cases} x = -\frac{2\pi}{3} + k2\pi \\ x = \pi + k2\pi \end{cases} (k \in \mathbb{Z}). \]

Đáp án: A.

Ví dụ 2: Tổng các nghiệm của phương trình \(3^{x^2 - 2x} = 81\) là:

  • A. 4
  • B. -4
  • C. -2
  • D. 2

Giải:

\(3^{x^2 - 2x} = 81 \Leftrightarrow 3^{x^2 - 2x} = 3^4 \Leftrightarrow x^2 - 2x - 4 = 0\).

Phương trình \(x^2 - 2x - 4 = 0\) có hệ số \(a, c\) trái dấu, nên có hai nghiệm phân biệt. Tổng hai nghiệm: \(-\frac{b}{a} = \frac{2}{1} = 2\).

Đáp án: D.

Ví dụ 3: Nghiệm của phương trình \(\log_{16}(x + 5) = \frac{1}{2}\) là:

  • A. 3
  • B. -1
  • C. -3
  • D. 27

Giải:

\(\log_{16}(x + 5) = \frac{1}{2} \Leftrightarrow x + 5 = 16^{\frac{1}{2}} = 4 \Leftrightarrow x = -1\).

Đáp án: B.

Ví dụ 4: Số nghiệm của phương trình \(\log_2(x - 4) = \log_2(x^2 - 5x + 4)\) là:

  • A. 1
  • B. 2
  • C. 0
  • D. 3

Giải:

\[ \log_2(x - 4) = \log_2(x^2 - 5x + 4) \Leftrightarrow \begin{cases} x - 4 = x^2 - 5x + 4 \\ x - 4 > 0 \end{cases} \Leftrightarrow \begin{cases} x^2 - 6x + 8 = 0 \\ x > 4 \end{cases} \Leftrightarrow \begin{cases} x = 2 \text{ hoặc } x = 4 \\ x > 4 \end{cases}. \]

Vậy phương trình vô nghiệm.

Đáp án: C.

Dạng 2: Câu trắc nghiệm đúng/sai

Ví dụ 5: Cho phương trình \(\sin^2 \left(2x + \frac{\pi}{4}\right) = \cos^2 \left(x + \frac{\pi}{2}\right)\).

  • a) Hạ bậc hai vế, ta được phương trình: \(\frac{1 + \cos \left(4x + \frac{\pi}{2}\right)}{2} = \frac{1 - \cos (2x + \pi)}{2}\). [Đúng/Sai]
  • b) Ta có: \(\cos (2x + \pi) = -\cos 2x\). [Đúng/Sai]
  • c) Phương trình đã cho đưa về dạng: \(\cos \left(4x + \frac{\pi}{2}\right) = \cos 2x\). [Đúng/Sai]
  • d) Nghiệm của phương trình đã cho là: \(x = -\frac{\pi}{4} + k\pi\) và \(x = \frac{\pi}{12} + k\frac{\pi}{3} (k \in \mathbb{Z})\). [Đúng/Sai]

Giải:

\[ \sin^2 \left(2x + \frac{\pi}{4}\right) = \cos^2 \left(x + \frac{\pi}{2}\right) \Leftrightarrow \frac{1 - \cos \left(4x + \frac{\pi}{2}\right)}{2} = \frac{1 + \cos (2x + \pi)}{2} \Leftrightarrow -\cos \left(4x + \frac{\pi}{2}\right) = \cos (2x + \pi). \]

Vì \(\cos (2x + \pi) = -\cos 2x\), nên:

\[ \cos \left(4x + \frac{\pi}{2}\right) = \cos 2x \Leftrightarrow \begin{cases} 4x + \frac{\pi}{2} = 2x + k2\pi \\ 4x + \frac{\pi}{2} = -2x + k2\pi \end{cases} \Leftrightarrow \begin{cases} x = -\frac{\pi}{4} + k\pi \\ x = -\frac{\pi}{12} + k\frac{\pi}{3} \end{cases}. \]

Đáp án: a) Sai, b) Đúng, c) Đúng, d) Sai.

Ví dụ 6: Cho bất phương trình \((3 - 2\sqrt{2})^{x^2 - 4x} > (3 + 2\sqrt{2})^{5 - 2x}\).

  • a) Ta có: \(3 + 2\sqrt{2} = (3 - 2\sqrt{2})^{-1}\). [Đúng/Sai]
  • b) Bất phương trình đã cho tương đương với bất phương trình: \(x^2 - 4x > 2x - 5\). [Đúng/Sai]
  • c) Số nghiệm nguyên của bất phương trình là 5. [Đúng/Sai]
  • d) Tổng các nghiệm nguyên của bất phương trình là 9. [Đúng/Sai]

Giải:

Vì \(3 + 2\sqrt{2} = (3 - 2\sqrt{2})^{-1}\), nên:

\[ (3 - 2\sqrt{2})^{x^2 - 4x} > (3 - 2\sqrt{2})^{2x - 5} \Leftrightarrow x^2 - 4x < 2x - 5 \Leftrightarrow x^2 - 6x + 5 < 0 \Leftrightarrow 1 < x < 5. \]

Nghiệm nguyên: \(x = 2, 3, 4\). Tổng: \(2 + 3 + 4 = 9\).

Đáp án: a) Đúng, b) Sai, c) Đúng, d) Đúng.

Ví dụ 7: Cho bất phương trình: \(\log_{\sqrt{2} - 1}(2x^2 - 2) \geq \log_{\sqrt{2} - 1}(5x + 5)\).

  • a) Ta có: \(0 < \sqrt{2} - 1 < 1\). [Đúng/Sai]
  • b) Bất phương trình đã cho tương đương với: \(\begin{cases} 2x^2 - 2 \leq 5x + 5 \\ 5x + 5 > 0 \end{cases}\). [Đúng/Sai]
  • c) Số nghiệm nguyên của bất phương trình là 2. [Đúng/Sai]
  • d) Nghiệm nguyên nhỏ nhất của bất phương trình là 0. [Đúng/Sai]

Giải:

Vì \(0 < \sqrt{2} - 1 < 1\), nên bất phương trình tương đương:

\[ \begin{cases} 2x^2 - 2 \leq 5x + 5 \\ 2x^2 - 2 > 0 \end{cases} \Leftrightarrow \begin{cases} 2x^2 - 5x - 7 \leq 0 \\ x^2 > 1 \end{cases} \Leftrightarrow 1 < x \leq \frac{7}{2}. \]

Nghiệm nguyên: \(x = 2, 3\). Nghiệm nhỏ nhất là 2.

Đáp án: a) Đúng, b) Đúng, c) Đúng, d) Sai.

Dạng 3: Câu trắc nghiệm trả lời ngắn

Ví dụ 8: Hàng ngày mực nước tại một cảng biển lên xuống theo thủy triều. Độ sâu \(h\) (m) của mực nước theo thời gian \(t\) (giờ) trong một ngày được cho bởi công thức:

\[ h = 16 + 7 \sin \left(\frac{\pi}{12} t\right), \text{với } 0 \leq t \leq 24. \]

Tìm thời điểm mà mực nước tại cảng là cao nhất.

Giải:

Do \(-1 \leq \sin \left(\frac{\pi}{12} t\right) \leq 1\), nên \(9 \leq h \leq 23\). Mực nước cao nhất là \(23 \, \text{m}\) khi:

\[ \sin \left(\frac{\pi}{12} t\right) = 1 \Leftrightarrow \frac{\pi}{12} t = \frac{\pi}{2} + k2\pi \Leftrightarrow t = 6 + 24k (k \in \mathbb{Z}). \]

Với \(0 \leq t \leq 24\), ta được \(t = 6\). Vậy thời điểm mực nước cao nhất là \(6\) giờ.

Ví dụ 9: Công thức định luật làm mát của Newton: \(kt = \ln \frac{T - S}{T_0 - S}\). Tính nhiệt độ của cốc trà sau 10 phút (làm tròn đến 1 chữ số thập phân).

Giải:

Thay \(t = 2 \, \text{phút} = \frac{1}{30} \, \text{giờ}\), \(T_0 = 96\), \(T = 90\), \(S = 24\):

\[\frac{1}{30} k = \ln \frac{90 - 24}{96 - 24} \Rightarrow k = 30 \ln \frac{11}{12}.\]

Sau 10 phút = \(\frac{1}{6} \, \text{giờ}\):

\[\frac{1}{6} k = \ln \frac{T - 24}{96 - 24} \Rightarrow 5 \ln \frac{11}{12} = \ln \frac{T - 24}{72} \Rightarrow \frac{T - 24}{72} = \left(\frac{11}{12}\right)^5.\]

\[T = 72 \cdot \left(\frac{11}{12}\right)^5 + 24 \approx 70.6^\circ \text{C}.\]

Câu 27:

\[ \cos^2 \left(\frac{\pi}{2} - x\right) = \sin^2 \left(3x + \frac{\pi}{4}\right) \Leftrightarrow \cos (\pi - 2x) = -\cos \left(6x + \frac{\pi}{2}\right) \Leftrightarrow \cos (\pi - 2x) = \cos \left(\frac{\pi}{2} - 6x\right). \]

\[ \Leftrightarrow \begin{cases} \pi - 2x = \frac{\pi}{2} - 6x + k2\pi \\ \pi - 2x = -\frac{\pi}{2} + 6x + k2\pi \end{cases} \Leftrightarrow \begin{cases} x = -\frac{\pi}{8} + k\frac{\pi}{2} \\ x = \frac{3\pi}{16} + k\frac{\pi}{4} \end{cases}. \]

Đáp án: a) Đúng, b) Sai, c) Sai, d) Sai.

Câu 28:

Vì \(\cos 2x = \sin \left(\frac{\pi}{2} - 2x\right)\), nên:

\[ \sin \left(\frac{\pi}{2} - 2x\right) = \sin \left(\frac{\pi}{4} - x\right) \Leftrightarrow \begin{cases} \frac{\pi}{2} - 2x = \frac{\pi}{4} - x + k2\pi \\ \frac{\pi}{2} - 2x = \pi - \left(\frac{\pi}{4} - x\right) + k2\pi \end{cases} \Leftrightarrow \begin{cases} x = \frac{\pi}{4} + k2\pi \\ x = -\frac{\pi}{12} + k\frac{2\pi}{3} \end{cases}. \]

Với \(x \in [0, \pi]\), nghiệm là \(x = \frac{\pi}{4}\), \(x = \frac{7\pi}{12}\). Tổng: \(\frac{\pi}{4} + \frac{7\pi}{12} = \frac{5\pi}{6}\).

Đáp án: a) Đúng, b) Sai, c) Đúng, d) Đúng.

Câu 29:

Dùng công thức biến đổi tổng thành tích:

\[ \sin 4x + \sin 2x = 2 \sin 3x \cos x, \quad \cos 4x + \cos 2x = 2 \cos 3x \cos x. \]

\[ 2 \sin 3x \cos x = 2 \cos 3x \cos x \Leftrightarrow \cos x (\sin 3x - \cos 3x) = 0 \Leftrightarrow \cos x = 0 \text{ hoặc } \sin 3x = \cos 3x. \]

\[ \cos x = 0 \Leftrightarrow x = \frac{\pi}{2} + k\pi. \]

\[ \sin 3x = \cos 3x \Leftrightarrow \tan 3x = 1 \Leftrightarrow 3x = \frac{\pi}{4} + k\pi \Leftrightarrow x = \frac{\pi}{12} + k\frac{\pi}{3}. \]

Đáp án: a) Sai, b) Sai, c) Đúng, d) Sai.

Câu 30:

Do \(-1 \leq \sin \left(\frac{\pi}{12} t\right) \leq 1\), nên \(6 \leq h \leq 22\). Chiều cao cao nhất \(22 \, \text{m}\) khi:

\[ \sin \left(\frac{\pi}{12} t\right) = 1 \Leftrightarrow \frac{\pi}{12} t = \frac{\pi}{2} + k2\pi \Leftrightarrow t = 6 + 24k. \]

Với \(0 \leq t \leq 24\), \(t = 6\). Chiều cao thấp nhất \(6 \, \text{m}\) khi:

\[ \sin \left(\frac{\pi}{12} t\right) = -1 \Leftrightarrow t = 18. \]

Xét \(h = 18\):

\[ \sin \left(\frac{\pi}{12} t\right) = \frac{1}{2} \Leftrightarrow \begin{cases} t = 2 + 24k \\ t = 10 + 24k \end{cases}. \]

Thời gian tàu vào cảng: từ \(2\) giờ đến \(10\) giờ.

Đáp án: a) Đúng, b) Sai, c) Sai.
