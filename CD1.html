1.A Tóm tắt kiến thức cần nhớ

1.1. Phương trình lượng giác cơ bản

a) Phương trình \(\sin x = m\)

  • Với \(|m| > 1\), phương trình vô nghiệm.
  • Với \(|m| \leq 1\), gọi \(\alpha \in \left[-\frac{\pi}{2}; \frac{\pi}{2}\right]\) sao cho \(\sin \alpha = m\). Khi đó: \[ \sin x = m \Leftrightarrow \sin x = \sin \alpha \Leftrightarrow \left\{ \begin{array}{l} x = \alpha + k 2\pi \\ x = \pi - \alpha + k 2\pi \end{array} (k \in \mathbb{Z}) \right. \]
  • Các trường hợp đặc biệt:
    • \(\sin x = 1 \Leftrightarrow x = \frac{\pi}{2} + k 2\pi\)
    • \(\sin x = -1 \Leftrightarrow x = -\frac{\pi}{2} + k 2\pi\)
    • \(\sin x = 0 \Leftrightarrow x = k \pi\)
  • Nếu \(x\) là góc đo bằng độ: \[ \sin x = \sin a^\circ \Leftrightarrow \left\{ \begin{array}{l} x = a^\circ + k 360^\circ \\ x = 180^\circ - a^\circ + k 360^\circ \end{array} (k \in \mathbb{Z}) \right. \]

b) Phương trình \(\cos x = m\)

  • Với \(|m| > 1\), phương trình vô nghiệm.
  • Với \(|m| \leq 1\), gọi \(\alpha \in [0; \pi]\) sao cho \(\cos \alpha = m\). Khi đó: \[ \cos x = m \Leftrightarrow \cos x = \cos \alpha \Leftrightarrow \left\{ \begin{array}{l} x = \alpha + k 2\pi \\ x = -\alpha + k 2\pi \end{array} (k \in \mathbb{Z}) \right. \]
  • Các trường hợp đặc biệt:
    • \(\cos x = 1 \Leftrightarrow x = k 2\pi\)
    • \(\cos x = -1 \Leftrightarrow x = \pi + k 2\pi\)
    • \(\cos x = 0 \Leftrightarrow x = \frac{\pi}{2} + k \pi\)
  • Nếu \(x\) là góc đo bằng độ: \[ \cos x = \cos a^\circ \Leftrightarrow \left\{ \begin{array}{l} x = a^\circ + k 360^\circ \\ x = -a^\circ + k 360^\circ \end{array} (k \in \mathbb{Z}) \right. \]

c) Phương trình \(\tan x = m\)

  • Gọi \(\alpha \in \left(-\frac{\pi}{2}; \frac{\pi}{2}\right)\) sao cho \(\tan \alpha = m\). Khi đó: \[ \tan x = m \Leftrightarrow \tan x = \tan \alpha \Leftrightarrow x = \alpha + k \pi \quad (k \in \mathbb{Z}) \]
  • Nếu \(x\) là góc đo bằng độ: \[ \tan x = \tan a^\circ \Leftrightarrow x = a^\circ + k 180^\circ \quad (k \in \mathbb{Z}) \]

d) Phương trình \(\cot x = m\)

  • Gọi \(\alpha \in (0; \pi)\) sao cho \(\cot \alpha = m\). Khi đó: \[ \cot x = m \Leftrightarrow \cot x = \cot \alpha \Leftrightarrow x = \alpha + k \pi \quad (k \in \mathbb{Z}) \]
  • Nếu \(x\) là góc đo bằng độ: \[ \cot x = \cot a^\circ \Leftrightarrow x = a^\circ + k 180^\circ \quad (k \in \mathbb{Z}) \]

1.2. Phương trình lượng giác đưa về dạng cơ bản

  • \(\sin f(x) = \sin g(x) \Leftrightarrow \left\{ \begin{array}{l} f(x) = g(x) + k 2\pi \\ f(x) = \pi - g(x) + k 2\pi \end{array} (k \in \mathbb{Z}) \right.\)
  • \(\cos f(x) = \cos g(x) \Leftrightarrow \left\{ \begin{array}{l} f(x) = g(x) + k 2\pi \\ f(x) = -g(x) + k 2\pi \end{array} (k \in \mathbb{Z}) \right.\)
  • Phương trình dạng \(\sin^2 u(x) = \sin^2 v(x)\), \(\cos^2 u(x) = \cos^2 v(x)\), hoặc \(\sin^2 u(x) = \cos^2 v(x)\): dùng công thức hạ bậc để đưa về \(\cos f(x) = \cos g(x)\).
  • Một số phương trình lượng giác: dùng công thức lượng giác và biến đổi để đưa về phương trình dạng tích \(A(x) \cdot B(x) = 0\).

1.3. Phương trình và bất phương trình mũ, lôgarit

a) Phương trình mũ

  • Với \(a > 0\), \(a \neq 1\):
    • \(a^{f(x)} = b \Leftrightarrow f(x) = \log_a b \quad (b > 0)\)
    • \(a^{f(x)} = a^{g(x)} \Leftrightarrow f(x) = g(x)\)

b) Phương trình lôgarit

  • Với \(a > 0\), \(a \neq 1\):
    • \(\log_a f(x) = b \Leftrightarrow f(x) = a^b\)
    • \(\log_a f(x) = \log_a g(x) \Leftrightarrow \left\{ \begin{array}{l} f(x) = g(x) \\ f(x) > 0 \text{ hoặc } g(x) > 0 \end{array} \right.\)

c) Bất phương trình mũ

  • Với \(a > 0\), \(a \neq 1\), xét \(a^{f(x)} > b\):
    • Nếu \(b \leq 0\), tập nghiệm là tập xác định của \(f(x)\).
    • Nếu \(b > 0\), \(a > 1\): \(f(x) > \log_a b\).
    • Nếu \(b > 0\), \(0 < a < 1\): \(f(x) < \log_a b\).
  • Xét \(a^{f(x)} > a^{g(x)}\):
    • Nếu \(a > 1\): \(f(x) > g(x)\).
    • Nếu \(0 < a < 1\): \(f(x) < g(x)\).

d) Bất phương trình lôgarit

  • Với \(a > 0\), \(a \neq 1\), xét \(\log_a f(x) > b\):
    • Nếu \(a > 1\): \(f(x) > a^b\).
    • Nếu \(0 < a < 1\): \(0 < f(x) < a^b\).
  • Xét \(\log_a f(x) > \log_a g(x)\):
    • Nếu \(a > 1\): \(f(x) > g(x) > 0\).
    • Nếu \(0 < a < 1\): \(0 < f(x) < g(x)\).

2. Ví dụ minh họa

Tóm tắt kiến thức

Phương trình \(\sin x = m\): Với \(|m| \leq 1\), gọi \(\alpha \in \left[-\frac{\pi}{2}; \frac{\pi}{2}\right]\) sao cho \(\sin \alpha = m\). Khi đó: \[ \sin x = m \Leftrightarrow \left\{ \begin{array}{l} x = \alpha + k 2\pi \\ x = \pi - \alpha + k 2\pi \end{array} (k \in \mathbb{Z}) \right. \]

Ví dụ 1: Nghiệm của phương trình \(\sin \left(x + \frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2}\) là:

  • A. \(x = -\frac{2\pi}{3} + k 2\pi\) và \(x = \pi + k 2\pi\)
  • B. \(x = -\frac{\pi}{3} + k 2\pi\) và \(x = \frac{\pi}{3} + k 2\pi\)
  • C. \(x = k 2\pi\) và \(x = \pi + k 2\pi\)
  • D. \(x = -\frac{\pi}{2} + k 2\pi\) và \(x = \frac{5\pi}{3} + k 2\pi\)

Lời giải:

Do \(\sin \left(-\frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2}\), ta có:

\[ \sin \left(x + \frac{\pi}{3}\right) = -\frac{\sqrt{3}}{2} \Leftrightarrow \sin \left(x + \frac{\pi}{3}\right) = \sin \left(-\frac{\pi}{3}\right) \] \[ \Leftrightarrow \left\{ \begin{array}{l} x + \frac{\pi}{3} = -\frac{\pi}{3} + k 2\pi \\ x + \frac{\pi}{3} = \pi - \left(-\frac{\pi}{3}\right) + k 2\pi \end{array} \right. \Leftrightarrow \left\{ \begin{array}{l} x = -\frac{2\pi}{3} + k 2\pi \\ x = \pi + k 2\pi \end{array} (k \in \mathbb{Z}) \right. \]

Chọn A.

Tóm tắt kiến thức

Phương trình mũ: \(a^{f(x)} = a^{g(x)} \Leftrightarrow f(x) = g(x)\).

Ví dụ 2: Tổng các nghiệm của phương trình \(3^{x^2 - 2x} = 81\) là:

  • A. 4
  • B. -4
  • C. -2
  • D. 2

Lời giải:

\[ 3^{x^2 - 2x} = 81 \Leftrightarrow 3^{x^2 - 2x} = 3^4 \Leftrightarrow x^2 - 2x - 4 = 0 \]

Phương trình \(x^2 - 2x - 4 = 0\) có hai nghiệm phân biệt (do \(\Delta = 4 + 16 = 20 > 0\)). Tổng các nghiệm là \(-\frac{b}{a} = \frac{2}{1} = 2\).

Chọn D.

Tóm tắt kiến thức

Phương trình lôgarit: \(\log_a f(x) = b \Leftrightarrow f(x) = a^b\).

Ví dụ 3: Nghiệm của phương trình \(\log_{16}(x + 5) = \frac{1}{2}\) là:

  • A. 3
  • B. -1
  • C. -3
  • D. 27

Lời giải:

\[ \log_{16}(x + 5) = \frac{1}{2} \Leftrightarrow x + 5 = 16^{\frac{1}{2}} = 4 \Leftrightarrow x = 4 - 5 = -1 \]

Chọn B.

Tóm tắt kiến thức

Phương trình lôgarit: \(\log_a f(x) = \log_a g(x) \Leftrightarrow \left\{ \begin{array}{l} f(x) = g(x) \\ f(x) > 0 \text{ hoặc } g(x) > 0 \end{array} \right.\)

Ví dụ 4: Số nghiệm của phương trình \(\log_2 (x - 4) = \log_2 (x^2 - 5x + 4)\) là:

  • A. 1
  • B. 2
  • C. 0
  • D. 3

Lời giải:

\[ \log_2 (x - 4) = \log_2 (x^2 - 5x + 4) \Leftrightarrow \left\{ \begin{array}{l} x - 4 = x^2 - 5x + 4 \\ x - 4 > 0 \end{array} \right. \] \[ \Leftrightarrow \left\{ \begin{array}{l} x^2 - 6x + 8 = 0 \\ x > 4 \end{array} \right. \Leftrightarrow \left\{ \begin{array}{l} x = 2 \text{ hoặc } x = 4 \\ x > 4 \end{array} \right. \]

Không có giá trị \(x\) thỏa mãn \(x > 4\). Vậy phương trình vô nghiệm.

Chọn C.

Tóm tắt kiến thức

Phương trình dạng \(\sin^2 u(x) = \cos^2 v(x)\): Dùng công thức hạ bậc: \[ \sin^2 \theta = \frac{1 - \cos 2\theta}{2}, \quad \cos^2 \theta = \frac{1 + \cos 2\theta}{2} \] để đưa về \(\cos f(x) = \cos g(x)\).

Ví dụ 5: Cho phương trình \(\sin^2 \left(2x + \frac{\pi}{4}\right) = \cos^2 \left(x + \frac{\pi}{2}\right)\). Xác định các khẳng định sau là đúng hay sai:

  • a) Hạ bậc hai vế, ta được phương trình: \(\frac{1 + \cos \left(4x + \frac{\pi}{2}\right)}{2} = \frac{1 - \cos (2x + \pi)}{2}\).
  • b) Ta có: \(\cos (2x + \pi) = -\cos 2x\).
  • c) Phương trình đã cho đưa về dạng: \(\cos \left(4x + \frac{\pi}{2}\right) = \cos 2x\).
  • d) Nghiệm của phương trình là: \(x = -\frac{\pi}{4} + k \pi\) và \(x = \frac{\pi}{12} + k \frac{\pi}{3}\).

Lời giải:

\[ \sin^2 \left(2x + \frac{\pi}{4}\right) = \cos^2 \left(x + \frac{\pi}{2}\right) \Leftrightarrow \frac{1 - \cos \left(4x + \frac{\pi}{2}\right)}{2} = \frac{1 + \cos (2x + \pi)}{2} \] \[ \Leftrightarrow -\cos \left(4x + \frac{\pi}{2}\right) = \cos (2x + \pi) \Leftrightarrow \cos \left(4x + \frac{\pi}{2}\right) = -\cos (2x + \pi) = \cos 2x \] \[ \Leftrightarrow \left\{ \begin{array}{l} 4x + \frac{\pi}{2} = 2x + k 2\pi \\ 4x + \frac{\pi}{2} = -2x + k 2\pi \end{array} \right. \Leftrightarrow \left\{ \begin{array}{l} x = -\frac{\pi}{4} + k \pi \\ x = \frac{\pi}{12} + k \frac{\pi}{3} \end{array} (k \in \mathbb{Z}) \right. \]

Kiểm tra các khẳng định:

  • a) Sai, vì hạ bậc đúng phải là \(\frac{1 - \cos \left(4x + \frac{\pi}{2}\right)}{2} = \frac{1 + \cos (2x + \pi)}{2}\).
  • b) Đúng, vì \(\cos (2x + \pi) = \cos (2x + \pi - 2\pi) = \cos 2x\).
  • c) Đúng.
  • d) Đúng.

Đáp án: a) S, b) Đ, c) Đ, d) S.

Tóm tắt kiến thức

Bất phương trình mũ: Với \(0 < a < 1\), \(a^{f(x)} > a^{g(x)} \Leftrightarrow f(x) < g(x)\).

Ví dụ 6: Cho bất phương trình \((3 - 2\sqrt{2})^{x^2 - 4x} > (3 + 2\sqrt{2})^{5 - 2x}\). Xác định các khẳng định sau là đúng hay sai:

  • a) Ta có: \(3 + 2\sqrt{2} = (3 - 2\sqrt{2})^{-1}\).
  • b) Bất phương trình đã cho tương đương với: \(x^2 - 4x > 2x - 5\).
  • c) Số nghiệm nguyên của bất phương trình là 5.
  • d) Tổng các nghiệm nguyên của bất phương trình là 9.

Lời giải:

Do \(3 + 2\sqrt{2} = (3 - 2\sqrt{2})^{-1}\), ta có:

\[ (3 - 2\sqrt{2})^{x^2 - 4x} > (3 - 2\sqrt{2})^{2x - 5} \]

Với \(0 < 3 - 2\sqrt{2} < 1\), bất phương trình trở thành:

\[ x^2 - 4x < 2x - 5 \Leftrightarrow x^2 - 6x + 5 < 0 \Leftrightarrow 1 < x < 5 \]

Các nghiệm nguyên: \(x = 2, 3, 4\). Tổng: \(2 + 3 + 4 = 9\).

Kiểm tra các khẳng định:

  • a) Đúng.
  • b) Sai, vì bất phương trình tương đương \(x^2 - 4x < 2x - 5\).
  • c) Sai, số nghiệm nguyên là 3.
  • d) Đúng.

Đáp án: a) Đ, b) S, c) S, d) Đ.

Tóm tắt kiến thức

Bất phương trình lôgarit: Với \(0 < a < 1\), \(\log_a f(x) > \log_a g(x) \Leftrightarrow 0 < f(x) < g(x)\).

Ví dụ 7: Cho bất phương trình \(\log_{\sqrt{2} - 1} (2x^2 - 2) \geq \log_{\sqrt{2} - 1} (5x + 5)\). Xác định các khẳng định sau là đúng hay sai:

  • a) Ta có: \(0 < \sqrt{2} - 1 < 1\).
  • b) Bất phương trình đã cho tương đương với: \(\left\{ \begin{array}{l} 2x^2 - 2 \leq 5x + 5 \\ 5x + 5 > 0 \end{array} \right.\).
  • c) Số nghiệm nguyên của bất phương trình là 2.
  • d) Nghiệm nguyên nhỏ nhất của bất phương trình là 0.

Lời giải:

Với \(0 < \sqrt{2} - 1 < 1\), ta có:

\[ \log_{\sqrt{2} - 1} (2x^2 - 2) \geq \log_{\sqrt{2} - 1} (5x + 5) \Leftrightarrow \left\{ \begin{array}{l} 2x^2 - 2 \leq 5x + 5 \\ 2x^2 - 2 > 0 \end{array} \right. \] \[ \Leftrightarrow \left\{ \begin{array}{l} 2x^2 - 5x - 7 \leq 0 \\ x^2 - 1 > 0 \end{array} \right. \Leftrightarrow 1 < x \leq \frac{7}{2} \]

Các nghiệm nguyên: \(x = 2, 3\). Nghiệm nguyên nhỏ nhất là 2.

Kiểm tra các khẳng định:

  • a) Đúng.
  • b) Sai, vì điều kiện đúng phải là \(2x^2 - 2 > 0\).
  • c) Đúng.
  • d) Sai, nghiệm nguyên nhỏ nhất là 2.

Đáp án: a) Đ, b) S, c) Đ, d) S.

Ví dụ 8: Hằng ngày, mực nước tại cảng theo thời gian \(t\) (giờ) được cho bởi công thức:

\[ h = 16 + 7 \sin \left(\frac{\pi}{12} t\right) \quad \text{với } 0 \leq t \leq 24 \]

Tìm thời điểm mà mực nước tại cảng cao nhất.

Lời giải:

Do \(-1 \leq \sin \left(\frac{\pi}{12} t\right) \leq 1\), ta có:

\[ 16 - 7 \leq h \leq 16 + 7 \Rightarrow 9 \leq h \leq 23 \]

Mực nước cao nhất là \(h = 23\) khi:

\[ \sin \left(\frac{\pi}{12} t\right) = 1 \Leftrightarrow \frac{\pi}{12} t = \frac{\pi}{2} + k 2\pi \Leftrightarrow t = 6 + 24k \quad (k \in \mathbb{Z}) \]

Với \(0 \leq t \leq 24\), ta có \(t = 6\). Vậy thời điểm mực nước cao nhất là \(t = 6\) giờ.

Phương trình lôgarit: \(\ln \frac{a}{b} = c \Leftrightarrow \frac{a}{b} = e^c\).

Ví dụ 9: Công thức định luật làm mát của Newton:

\[ kt = \ln \frac{T - S}{T_0 - S} \]

Thay \(t = 2\) phút = \(\frac{1}{30}\) giờ, \(T_0 = 96\), \(T = 90\), \(S = 24\). Tính nhiệt độ \(T\) sau 10 phút.

Lời giải:

Thay \(t = \frac{1}{30}\), \(T_0 = 96\), \(T = 90\), \(S = 24\):

\[ \frac{1}{30} k = \ln \frac{90 - 24}{96 - 24} = \ln \frac{66}{72} = \ln \frac{11}{12} \] \[ k = 30 \ln \frac{11}{12} \]

Sau 10 phút = \(\frac{1}{6}\) giờ:

\[ \frac{1}{6} k = \ln \frac{T - 24}{96 - 24} \Rightarrow \frac{1}{6} \cdot 30 \ln \frac{11}{12} = \ln \frac{T - 24}{72} \] \[ 5 \ln \frac{11}{12} = \ln \frac{T - 24}{72} \Rightarrow \frac{T - 24}{72} = \left(\frac{11}{12}\right)^5 \] \[ T = 72 \cdot \left(\frac{11}{12}\right)^5 + 24 \approx 70.6^\circ \text{C} \]

Vậy nhiệt độ sau 10 phút khoảng \(70.6^\circ \text{C}\).

Dạng 1: Trắc Nghiệm Nhiều Phương Án Lựa Chọn

Tóm tắt kiến thức: Phương trình lượng giác

Phương trình \(\sin x = 0\) có nghiệm \(x = k\pi\), \(k \in \mathbb{Z}\).

Câu 1:

Các nghiệm của phương trình \(\sin \left(\frac{\pi}{5} - x\right) = 0\) là:

A. \(x = -\frac{\pi}{5} + k\pi\), \(k \in \mathbb{Z}\)
B. \(x = \frac{2\pi}{5} + k\pi\), \(k \in \mathbb{Z}\)
C. \(x = -\frac{\pi}{5} + k2\pi\), \(k \in \mathbb{Z}\)
D. \(x = \frac{\pi}{5} + k\pi\), \(k \in \mathbb{Z}\)

Chọn đáp án:

Xem lời giải

Lời giải:
\(\sin \left(\frac{\pi}{5} - x\right) = 0 \Rightarrow \frac{\pi}{5} - x = k\pi \Rightarrow x = \frac{\pi}{5} - k\pi = \frac{\pi}{5} + k\pi\), \(k \in \mathbb{Z}\).
Đáp án: D.

Tóm tắt kiến thức: Phương trình lượng giác

Phương trình \(\sin ax = b\) có nghiệm nếu \(|b| \leq 1\), nghiệm: \(x = \frac{\arcsin b}{a} + k\frac{2\pi}{a}\) hoặc \(x = \frac{\pi - \arcsin b}{a} + k\frac{2\pi}{a}\).

Câu 2:

Các nghiệm của phương trình \(2\sin 3x + \sqrt{2} = 0\) là:

A. \(x = \frac{\pi}{12} + k\frac{2\pi}{3}\) và \(x = \frac{3\pi}{12} + k\frac{2\pi}{3}\), \(k \in \mathbb{Z}\)
B. \(x = -\frac{\pi}{12} + k\frac{2\pi}{3}\) và \(x = \frac{5\pi}{12} + k\frac{2\pi}{3}\), \(k \in \mathbb{Z}\)
C. \(x = \frac{\pi}{12} + k\frac{2\pi}{3}\) và \(x = -\frac{\pi}{12} + k\frac{2\pi}{3}\), \(k \in \mathbb{Z}\)
D. \(x = -\frac{\pi}{12} + k\frac{2\pi}{3}\) và \(x = \frac{3\pi}{12} + k\frac{2\pi}{3}\), \(k \in \mathbb{Z}\)

Chọn đáp án:

Xem lời giải

Lời giải:
\(2\sin 3x + \sqrt{2} = 0 \Rightarrow \sin 3x = -\frac{\sqrt{2}}{2} = -\sin \frac{\pi}{4} = \sin \left(-\frac{\pi}{4}\right)\).

Chọn đáp án:

Xem lời giải

Số nghiệm nguyên của bất phương trình \(\log_5 (2x - 3) < \log_{25} x^2\) là:

B. 2
C. 0
D. Vô số

Chọn đáp án:

Xem lời giải

Lời giải:
Điều kiện: \(2x - 3 > 0 \Rightarrow x > \frac{3}{2}\), \(x^2 > 0 \Rightarrow x \neq 0\).
