1.A Tóm tắt kiến thức cần nhớ

1.1. Vectơ

1.1.1. Vectơ và các khái niệm

\[ \text{Vectơ là một đoạn thẳng có hướng.} \] \[ \text{Giá của vectơ là đường thẳng đi qua hai đầu mút; độ dài là khoảng cách giữa hai đầu mút.} \] \[ \text{Hai vectơ cùng phương nếu giá song song hoặc trùng nhau; bằng nhau nếu cùng hướng và độ dài.} \] \[ \text{Vectơ-không } \vec{0} \text{ có điểm đầu và cuối trùng nhau; hai vectơ đối nhau nếu ngược hướng và cùng độ dài.} \]

1.1.2. Các phép toán vectơ

\[ \text{Tổng } \vec{a} + \vec{b}: \text{ Lấy điểm } A, \text{ vẽ } \overrightarrow{AB} = \vec{a}, \overrightarrow{BC} = \vec{b}, \text{ thì } \overrightarrow{AC} = \vec{a} + \vec{b}. \] \[ \text{Hiệu } \vec{a} - \vec{b} = \vec{a} + (-\vec{b}). \] \[ \text{Quy tắc hình bình hành: Nếu } ABCD \text{ là hình bình hành thì } \overrightarrow{AB} + \overrightarrow{AD} = \overrightarrow{AC}. \] \[ \text{Quy tắc hình hộp: Nếu } ABCD.A'B'C'D' \text{ là hình hộp thì } \overrightarrow{AB} + \overrightarrow{AD} + \overrightarrow{AA'} = \overrightarrow{AC'}. \] \[ \text{Quy tắc hiệu: } \overrightarrow{OA} - \overrightarrow{OB} = \overrightarrow{BA}. \] \[ \text{Tích số thực } k \text{ với } \vec{a} \neq \vec{0}: k\vec{a} \text{ có độ dài } |k| \cdot |\vec{a}|, \text{ cùng phương } \vec{a} \text{ nếu } k > 0, \text{ ngược phương nếu } k < 0. \] \[ k\vec{a} = \vec{0} \text{ khi và chỉ khi } k = 0 \text{ hoặc } \vec{a} = \vec{0}. \] \[ k(\vec{a} + \vec{b}) = k\vec{a} + k\vec{b}; \quad k(\vec{a} - \vec{b}) = k\vec{a} - k\vec{b}; \quad (h + k)\vec{a} = h\vec{a} + k\vec{a}; \quad h(k\vec{a}) = (hk)\vec{a}. \] \[ 1\vec{a} = \vec{a}; \quad (-1)\vec{a} = -\vec{a}. \] \[ \text{Hai vectơ } \vec{a}, \vec{b} \neq \vec{0} \text{ cùng phương khi và chỉ khi } \exists k \neq 0 \text{ sao cho } \vec{a} = k\vec{b}. \] \[ \text{Nếu } I \text{ là trung điểm } AB \text{ thì } \overrightarrow{IA} + \overrightarrow{IB} = \vec{0}. \] \[ \text{Nếu } G \text{ là trọng tâm tam giác } ABC \text{ thì } \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} = \vec{0}. \]

1.1.3. Tích vô hướng

\[ \vec{a} \cdot \vec{b} = \vec{b} \cdot \vec{a}; \quad \vec{a} \cdot (\vec{b} + \vec{c}) = \vec{a} \cdot \vec{b} + \vec{a} \cdot \vec{c}; \quad (k\vec{a}) \cdot \vec{b} = k(\vec{a} \cdot \vec{b}) = \vec{a} \cdot (k\vec{b}). \] \[ \vec{a}^2 = \vec{a} \cdot \vec{a} \geq 0; \quad \vec{a}^2 = 0 \Leftrightarrow \vec{a} = \vec{0}. \]

1.2. Phương pháp tọa độ trong không gian

1.2.1. Tọa độ của vectơ

\[ \overrightarrow{OM} = (a; b; c) \Leftrightarrow M(a; b; c). \] \[ \text{Tọa độ vectơ } \vec{u} \text{ là tọa độ điểm } A \text{ sao cho } \overrightarrow{OA} = \vec{u}. \] \[ \vec{u} = (a; b; c) \Leftrightarrow \vec{u} = a\vec{i} + b\vec{j} + c\vec{k}. \] \[ \vec{a} = (x_1; y_1; z_1), \vec{b} = (x_2; y_2; z_2): \vec{a} = \vec{b} \Leftrightarrow x_1 = x_2, y_1 = y_2, z_1 = z_2. \] \[ \overrightarrow{AB} = (x_B - x_A; y_B - y_A; z_B - z_A). \]

1.2.2. Biểu thức tọa độ các phép toán vectơ

\[ \vec{u} = (x_1; y_1; z_1), \vec{v} = (x_2; y_2; z_2): \quad \vec{u} + \vec{v} = (x_1 + x_2; y_1 + y_2; z_1 + z_2). \] \[ \vec{u} - \vec{v} = (x_1 - x_2; y_1 - y_2; z_1 - z_2). \] \[ m\vec{u} = (m x_1; m y_1; m z_1), \quad m \in \mathbb{R}. \] \[ \vec{u} \cdot \vec{v} = x_1 x_2 + y_1 y_2 + z_1 z_2. \] \[ [\vec{u}, \vec{v}] = \left( \begin{vmatrix} y_1 & z_1 \\ y_2 & z_2 \end{vmatrix}, \begin{vmatrix} z_1 & x_1 \\ z_2 & x_2 \end{vmatrix}, \begin{vmatrix} x_1 & y_1 \\ x_2 & y_2 \end{vmatrix} \right) = (y_1 z_2 - y_2 z_1; z_1 x_2 - z_2 x_1; x_1 y_2 - x_2 y_1). \] \[ \text{Hai vectơ } \vec{u}, \vec{v} \neq \vec{0} \text{ cùng phương } \Leftrightarrow \exists m: x_1 = m x_2, y_1 = m y_2, z_1 = m z_2. \] \[ |\vec{a}| = \sqrt{x^2 + y^2 + z^2}, \quad \vec{a} = (x; y; z). \] \[ AB = |\overrightarrow{AB}| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}. \] \[ \cos(\vec{u}, \vec{v}) = \frac{\vec{u} \cdot \vec{v}}{|\vec{u}| \cdot |\vec{v}|} = \frac{x_1 x_2 + y_1 y_2 + z_1 z_2}{\sqrt{x_1^2 + y_1^2 + z_1^2} \cdot \sqrt{x_2^2 + y_2^2 + z_2^2}}. \] \[ \text{Trung điểm } M \text{ của } AB: x_M = \frac{x_A + x_B}{2}, y_M = \frac{y_A + y_B}{2}, z_M = \frac{z_A + z_B}{2}. \] \[ \text{Trọng tâm } G \text{ tam giác } ABC: x_G = \frac{x_A + x_B + x_C}{3}, y_G = \frac{y_A + y_B + y_C}{3}, z_G = \frac{z_A + z_B + z_C}{3}. \]

1.2.3. Phương trình mặt phẳng

\[ \text{Vectơ pháp tuyến } \vec{n} = [\vec{a}, \vec{b}], \text{ với } \vec{a}, \vec{b} \text{ là cặp vectơ chỉ phương của mặt phẳng.} \] \[ \text{Mặt phẳng qua } I(x_0; y_0; z_0), \text{ pháp tuyến } \vec{n} = (a; b; c): a x + b y + c z + d = 0, \quad d = -a x_0 - b y_0 - c z_0. \] \[ \text{Mặt phẳng qua } A(a; 0; 0), B(0; b; 0), C(0; 0; c), \text{ với } abc \neq 0: \frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1. \] \[ \text{Hai mặt phẳng } (P_1): A_1 x + B_1 y + C_1 z + D_1 = 0, (P_2): A_2 x + B_2 y + C_2 z + D_2 = 0, \text{ với } \vec{n}_1 = (A_1; B_1; C_1), \vec{n}_2 = (A_2; B_2; C_2). \] \[ (P_1) \parallel (P_2) \Leftrightarrow \exists k \neq 0: \vec{n}_1 = k \vec{n}_2, D_1 \neq k D_2. \] \[ (P_1) \perp (P_2) \Leftrightarrow A_1 A_2 + B_1 B_2 + C_1 C_2 = 0. \] \[ \text{Khoảng cách từ } M_0(x_0; y_0; z_0) \text{ đến } (P): A x + B y + C z + D = 0: \quad d(M_0, (P)) = \frac{|A x_0 + B y_0 + C z_0 + D|}{\sqrt{A^2 + B^2 + C^2}}. \]

1.2.4. Phương trình đường thẳng

\[ \text{Vectơ chỉ phương } \vec{u} \neq \vec{0} \text{ có giá song song hoặc trùng với đường thẳng } \Delta. \] \[ \text{Phương trình đường thẳng qua } (x_0; y_0; z_0), \text{ chỉ phương } \vec{u} = (a; b; c): \frac{x - x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c}. \] \[ \text{Hai đường thẳng } \Delta_1, \Delta_2 \text{ qua } M_1, M_2, \text{ chỉ phương } \vec{u}_1, \vec{u}_2: \] \[ \Delta_1 \parallel \Delta_2 \Leftrightarrow [\vec{u}_1, \vec{u}_2] = \vec{0}, [\vec{u}_1, \overrightarrow{M_1 M_2}] \neq \vec{0}. \] \[ \Delta_1 \text{ cắt } \Delta_2 \Leftrightarrow [\vec{u}_1, \vec{u}_2] \neq \vec{0}, [\vec{u}_1, \vec{u}_2] \cdot \overrightarrow{M_1 M_2} = 0. \] \[ \Delta_1 \text{ chéo } \Delta_2 \Leftrightarrow [\vec{u}_1, \vec{u}_2] \cdot \overrightarrow{M_1 M_2} \neq 0. \]

1.2.5. Phương trình mặt cầu

\[ \text{Mặt cầu tâm } I(a; b; c), \text{ bán kính } R: (x - a)^2 + (y - b)^2 + (z - c)^2 = R^2. \] \[ x^2 + y^2 + z^2 - 2a x - 2b y - 2c z + d = 0 \text{ là mặt cầu nếu } a^2 + b^2 + c^2 - d > 0, \text{ tâm } I(a; b; c), R = \sqrt{a^2 + b^2 + c^2 - d}. \]

1.2.6. Góc

\[ \text{Góc giữa hai đường thẳng } \Delta_1, \Delta_2 \text{ với } \vec{u}_1 = (a_1; b_1; c_1), \vec{u}_2 = (a_2; b_2; c_2): \cos(\Delta_1, \Delta_2) = \frac{|a_1 a_2 + b_1 b_2 + c_1 c_2|}{\sqrt{a_1^2 + b_1^2 + c_1^2} \cdot \sqrt{a_2^2 + b_2^2 + c_2^2}}. \] \[ \Delta_1 \perp \Delta_2 \Leftrightarrow a_1 a_2 + b_1 b_2 + c_1 c_2 = 0. \] \[ \text{Góc giữa đường thẳng } \Delta \text{ và mặt phẳng } (P), \text{ với } \vec{u} = (a_1; b_1; c_1), \vec{n} = (a_2; b_2; c_2): \sin(\Delta, (P)) = \frac{|a_1 a_2 + b_1 b_2 + c_1 c_2|}{\sqrt{a_1^2 + b_1^2 + c_1^2} \cdot \sqrt{a_2^2 + b_2^2 + c_2^2}}. \] \[ \text{Góc giữa hai mặt phẳng } (P_1), (P_2) \text{ với } \vec{n}_1 = (A_1; B_1; C_1), \vec{n}_2 = (A_2; B_2; C_2): \cos((P_1), (P_2)) = \frac{|A_1 A_2 + B_1 B_2 + C_1 C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2} \cdot \sqrt{A_2^2 + B_2^2 + C_2^2}}. \]

2. Ví dụ minh họa

Tóm tắt kiến thức

Tích vô hướng: \( \vec{u} \cdot \vec{v} = |\vec{u}| \cdot |\vec{v}| \cdot \cos(\vec{u}, \vec{v}) \).

Ví dụ 1: Cho hai vectơ \( \vec{u}, \vec{v} \) có \( |\vec{u}| = 2 \), \( |\vec{v}| = 3 \), và \( (\vec{u}, \vec{v}) = 60^\circ \). Khi đó, \( \vec{u} \cdot \vec{v} \) bằng:

  • A. 3
  • B. 6
  • C. \( 3\sqrt{3} \)
  • D. 12

Lời giải:

\( \vec{u} \cdot \vec{v} = |\vec{u}| \cdot |\vec{v}| \cdot \cos 60^\circ = 2 \cdot 3 \cdot \frac{1}{2} = 3 \).

Đáp án: A

Tóm tắt kiến thức

Phương trình mặt cầu: \( (x - a)^2 + (y - b)^2 + (z - c)^2 = R^2 \), bán kính \( R \).

Ví dụ 2: Trong không gian \( Oxyz \), mặt cầu \( (S): (x - 2)^2 + (y - 3)^2 + (z - 4)^2 = 9 \) có bán kính bằng:

  • A. 81
  • B. 9
  • C. 3
  • D. 6

Lời giải:

Bán kính mặt cầu: \( R = \sqrt{9} = 3 \).

Đáp án: C

Tóm tắt kiến thức

Khoảng cách từ điểm đến điểm: \( AB = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2} \).

Ví dụ 3: Một tháp kiểm soát không lưu cao 80 m sử dụng radar phạm vi 500 km, đặt trên đỉnh tháp. Chọn hệ tọa độ \( Oxyz \) với gốc \( O \) tại chân tháp, mặt phẳng \( Oxy \) trùng mặt đất, trục \( Ox \) hướng tây, \( Oy \) hướng nam, \( Oz \) hướng lên. Một máy bay tại \( A \), cách mặt đất 10 km, cách 300 km về phía đông và 200 km về phía bắc so với tháp.

Xác định các phát biểu đúng/sai:

  • a) Radar ở vị trí có tọa độ \( (0; 0; 0) \).
  • b) Vị trí \( A \) có tọa độ \( (300; 200; 10) \).
  • c) Khoảng cách từ máy bay đến radar khoảng 360.69 km (làm tròn đến hàng phần trăm).
  • d) Radar không phát hiện được máy bay tại \( A \).

Lời giải:

- a) Radar tại đỉnh tháp: \( (0; 0; 0.08) \) (80 m = 0.08 km). Sai.

- b) Máy bay cách 300 km đông (\( x = -300 \)), 200 km bắc (\( y = -200 \)), cao 10 km (\( z = 10 \)): \( A(-300; -200; 10) \). Sai.

- c) Khoảng cách: \( \sqrt{(-300 - 0)^2 + (-200 - 0)^2 + (10 - 0.08)^2} \approx 360.69 \) km. Đúng.

- d) Vì 360.69 < 500, radar phát hiện được máy bay. Sai.

Đáp án: a) Sai, b) Sai, c) Đúng, d) Sai

Tóm tắt kiến thức

Mặt cầu tâm \( I(a; b; c) \), bán kính \( R \): điểm \( M \) nằm trong nếu \( IM < R \), ngoài nếu \( IM > R \).

Ví dụ 4: Trong không gian \( Oxyz \), trạm thu phát sóng có vùng phủ sóng là mặt cầu \( (S) \) tâm \( I(-1; -3; -7) \), bán kính 3 km. Xác định các phát biểu đúng/sai:

  • a) Phương trình mặt cầu: \( (x + 1)^2 + (y + 3)^2 + (z + 7)^2 = 9 \).
  • b) Điểm \( A(2; 2; 7) \) nằm ngoài mặt cầu \( (S) \).
  • c) Người dùng tại \( (2; 2; 7) \) có thể sử dụng dịch vụ.
  • d) Người dùng tại \( (5; 6; 7) \) không thể sử dụng dịch vụ.

Lời giải:

- a) Mặt cầu tâm \( I(-1; -3; -7) \), \( R = 3 \): \( (x + 1)^2 + (y + 3)^2 + (z + 7)^2 = 9 \). Đúng.

- b) \( IA = \sqrt{(2 + 1)^2 + (2 + 3)^2 + (7 + 7)^2} = \sqrt{218} \approx 14.76 > 3 \). Đúng.

- c) Vì \( IA > 3 \), điểm \( A \) ngoài mặt cầu, không sử dụng được dịch vụ. Sai.

- d) \( IB = \sqrt{(5 + 1)^2 + (6 + 3)^2 + (7 + 7)^2} = \sqrt{313} \approx 17.69 > 3 \). Không sử dụng được. Đúng.

Đáp án: a) Đúng, b) Đúng, c) Sai, d) Đúng

Tóm tắt kiến thức

Góc giữa hai vectơ: \( \cos(\vec{u}, \vec{v}) = \frac{\vec{u} \cdot \vec{v}}{|\vec{u}| \cdot |\vec{v}|} \).

Ví dụ 5: Trong không gian \( Oxyz \), cho tam giác \( ABC \) với \( A(1; -1; 3) \), \( B(-1; -1; 2) \), \( C(-3; -2; 2) \). Tính \( \cos \angle ABC \).

Lời giải:

\( \overrightarrow{BA} = (1 - (-1); -1 - (-1); 3 - 2) = (2; 0; 1) \), \( \overrightarrow{BC} = (-3 - (-1); -2 - (-1); 2 - 2) = (-2; -1; 0) \).

\( \overrightarrow{BA} \cdot \overrightarrow{BC} = 2 \cdot (-2) + 0 \cdot (-1) + 1 \cdot 0 = -4 \).

\( |\overrightarrow{BA}| = \sqrt{2^2 + 0^2 + 1^2} = \sqrt{5} \), \( |\overrightarrow{BC}| = \sqrt{(-2)^2 + (-1)^2 + 0^2} = \sqrt{5} \).

\( \cos \angle ABC = \frac{-4}{\sqrt{5} \cdot \sqrt{5}} = -0.8 \).

Đáp án: -0.8

Phần I: Trắc Nghiệm Nhiều Lựa Chọn

Tóm tắt kiến thức: Tứ diện và vector

Trong tứ diện \(ABCD\), trọng tâm \(G\) của tam giác \(ABC\) thỏa mãn \( \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} = \overrightarrow{0} \). Các tính chất vector trong không gian được sử dụng để kiểm tra các đẳng thức.

Câu 1:

Cho tứ diện \(ABCD\). Lấy \(G\) là trọng tâm của tam giác \(ABC\). Phát biểu nào sau đây là sai?

A. \( \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} = \overrightarrow{0} \)
B. \( \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} + \overrightarrow{GD} = \overrightarrow{0} \)
C. \( \overrightarrow{GD} - \overrightarrow{GA} = \overrightarrow{AD} \)
D. \( \overrightarrow{DA} + \overrightarrow{DB} + \overrightarrow{DC} = 3 \overrightarrow{DG} \)

Chọn đáp án:

Xem lời giải

Lời giải:
- A: Đúng, vì \(G\) là trọng tâm tam giác \(ABC\), nên \( \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} = \overrightarrow{0} \).
- B: Sai, vì \( \overrightarrow{GA} + \overrightarrow{GB} + \overrightarrow{GC} + \overrightarrow{GD} \) không nhất thiết bằng \(\overrightarrow{0}\), trừ khi \(G\) là trọng tâm của tứ diện.
- C: Đúng, vì \( \overrightarrow{GD} - \overrightarrow{GA} = \overrightarrow{GD} + \overrightarrow{AG} = \overrightarrow{AD} \).
- D: Đúng, vì \( \overrightarrow{DA} + \overrightarrow{DB} + \overrightarrow{DC} = 3 \overrightarrow{DG} \) (tính chất trọng tâm).
Đáp án: B.

Tóm tắt kiến thức: Tọa độ điểm

Trong không gian \(Oxyz\), vector \(\overrightarrow{OM} = x \vec{i} + y \vec{j} + z \vec{k}\) tương ứng với điểm \(M(x; y; z)\).

Câu 2:

Trong không gian \(Oxyz\), cho điểm \(M\) thỏa mãn \(\overrightarrow{OM} = 2 \vec{i} + 3 \vec{j} - 4 \vec{k}\). Tọa độ của điểm \(M\) là:

A. \((-4; 3; 2)\)
B. \((2; 3; -4)\)
C. \((3; -4; 2)\)
D. \((-2; -3; 4)\)

Chọn đáp án:

Xem lời giải

Lời giải:
\(\overrightarrow{OM} = 2 \vec{i} + 3 \vec{j} - 4 \vec{k}\) tương ứng với điểm \(M(2; 3; -4)\).
Đáp án: B.

Tóm tắt kiến thức: Hiệu hai vector

Cho hai vector \(\vec{u} = (u_1; u_2; u_3)\), \(\vec{v} = (v_1; v_2; v_3)\), thì \(\vec{u} - \vec{v} = (u_1 - v_1; u_2 - v_2; u_3 - v_3)\).

Câu 3:

Trong không gian \(Oxyz\), cho hai vector \(\vec{u} = (3; 2; -1)\), \(\vec{v} = (5; -4; 2)\). Tọa độ của vector \(\vec{u} - \vec{v}\) là:

A. \((-2; 6; -3)\)
B. \((-2; -2; -3)\)
C. \((2; 6; -3)\)
D. \((2; 2; 1)\)

Chọn đáp án:

Xem lời giải

Lời giải:
\(\vec{u} - \vec{v} = (3 - 5; 2 - (-4); -1 - 2) = (-2; 6; -3)\).
Đáp án: A.

Tóm tắt kiến thức: Nhân vector với số

Cho vector \(\vec{u} = (u_1; u_2; u_3)\), thì \(k \vec{u} = (k u_1; k u_2; k u_3)\).

Khi gắn hệ tọa độ \(Oxyz\) (đơn vị trên mỗi trục tính theo mét) vào một căn nhà sao cho nền nhà thuộc mặt phẳng \((Oxy)\), người ta coi mỗi mái nhà là một phần của mặt phẳng và thấy ba vị trí \(A, B, C\) ở mái nhà bên phải lần lượt có tọa độ \((2; 0; 4)\), \((4; 0; 3)\), và \((4; 9; 3)\). Góc giữa mái nhà bên phải và nền nhà bằng bao nhiêu độ (làm tròn kết quả đến hàng đơn vị)?

Nhập đáp án:

Xem lời giải

Lời giải:
Vectơ: \(\overrightarrow{AB} = (4-2; 0-0; 3-4) = (2; 0; -1)\), \(\overrightarrow{AC} = (4-2; 9-0; 3-4) = (2; 9; -1)\).
Vectơ pháp tuyến của \((ABC)\): \([\overrightarrow{AB}, \overrightarrow{AC}] = (1; 0; 2)\).
Vectơ pháp tuyến của \((Oxy)\): \(\vec{n}_2 = (0; 0; 1)\).
\(\cos \theta = \frac{|1 \cdot 0 + 0 \cdot 0 + 2 \cdot 1|}{\sqrt{1^2 + 0^2 + 2^2} \cdot \sqrt{0^2 + 0^2 + 1^2}} = \frac{2}{\sqrt{5}}\).
\(\theta \approx \arccos(2/\sqrt{5}) \approx 27^\circ\).
Đáp án: 27.

Tóm tắt kiến thức: Góc giữa đường thẳng và mặt phẳng

- Sin góc: \(\sin \theta = \frac{|\vec{u} \cdot \vec{n}|}{|\vec{u}| |\vec{n}|}\), với \(\vec{u}\) là vectơ chỉ phương, \(\vec{n}\) là vectơ pháp tuyến.

Câu 34:

Khi gắn hệ tọa độ \(Oxyz\) (đơn vị trên mỗi trục tính theo kilômét) vào một sân bay, mặt phẳng \((Oxy)\) trùng với mặt sân bay. Một máy bay bay qua hai điểm \(A(5; 0; 5)\) và \(B(10; 10; 2)\). Góc giữa đường bay \(AB\) và mặt sân bay bằng bao nhiêu độ (làm tròn kết quả đến hàng đơn vị)?

Nhập đáp án:

Xem lời giải

Lời giải:
Vectơ chỉ phương của \(AB\): \(\vec{u} = (10-5; 10-0; 2-5) = (5; 10; -3)\).
Vectơ

Vectơ pháp tuyến của \((Oxy)\): \(\vec{n} = (0; 0; 1)\).
\(\sin \theta = \frac{|5 \cdot 0 + 10 \cdot 0 + (-3) \cdot 1|}{\sqrt{5^2 + 10^2 + (-3)^2} \cdot \sqrt{0^2 + 0^2 + 1^2}} = \frac{3}{\sqrt{134}}\).

Câu 35:

Trong không gian \(Oxyz\), cho đường thẳng \(AB: \frac{x-5}{5} = \frac{y}{10} = \frac{z-5}{-2}\). Gọi \(M\) là giao điểm của đường thẳng \(AB\) với mặt phẳng \((Oxy)\). Nếu tọa độ của \(M\) là \((a; b; 0)\), giá trị của \(a + b\) bằng bao nhiêu?

Nhập đáp án:

Xem lời giải

Lời giải:
Phương trình đường thẳng: \(x = 5t + 5\), \(y = 10t\), \(z = -2t + 5\).
\(M \in (Oxy) \Rightarrow z = 0 \Rightarrow -2t + 5 = 0 \Rightarrow t = \frac{5}{2}\).
Tọa độ \(M\): \(x = 5 \cdot \frac{5}{2} + 5 = 17.5\), \(y = 10 \cdot \frac{5}{2} = 25\), \(z = 0\).
Vậy \(M(17.5; 25; 0)\), \(a + b = 17.5 + 25 = 42.5\).
Đáp án: 42.5.
